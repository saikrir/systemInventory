## Following Attributes will be tracked


1. systemID (integer that identifies the system)
2. systemName (char *, 64 length)
3. systemVendor (char *, 64 length)
4. cpuType (char *, 24 length)
5. clockSpeedGHZ (float, clock speed in Ghz)
6. memoryCapacityGB (float, memory in GB)
7. diskCapacity (float, capacity in GB)
8. os (char *, 24 length)
